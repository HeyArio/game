Rigging guide — layers & pivots

Every group below is a named <g id> in arbi-full.svg. Suggested transform-origins:

  • #arbi-body-rig — whole-body float bob. Translate Y ±4px, 3s ease.
  • #arbi-glow — antenna pulse. Opacity 0.35→1, 2s. Pivot 110,13.
  • #arbi-eye-left / #arbi-eye-right — blink. scaleY 1→0.08. Pivots 96,90 / 124,90.
  • #arbi-mouth — talk / smile. Morph the d path.
  • #arbi-head — tilt for "thinking". Rotate ±8°. Pivot 110,92.
  • #arbi-arm-right — raise / swing. Rotate. Pivot 142,150.
  • #arbi-gavel — "verdict" tap. Rotate -25°→0. Pivot 170,125.
  • #arbi-arm-left — wave / gesture. Rotate. Pivot 78,150.
  • #arbi-cheek-left / -right — blush fade-in on "happy".
  • #arbi-emblem — scales-of-justice; keep static or shimmer on verdict.

Workflow: open arbi-full.svg in Figma or Illustrator (layers are preserved) → export to Lottie via the LottieFiles plugin, or hand off to Rive / After Effects. For lightweight web use, the CSS in this file's <style> block is a ready starting point for an idle loop.
